环境变量可能包含敏感信息。本篇分析 Claude Code 如何隔离环境变量，防止泄露。

## 环境变量风险

### 敏感信息泄露

```bash
# 用户环境变量
AWS_ACCESS_KEY_ID=AKIAIOSFODNN7EXAMPLE
AWS_SECRET_ACCESS_KEY=wJalrXUtnFEMI/K7MDENG
ANTHROPIC_API_KEY=sk-ant-xxx
DATABASE_URL=postgres://user:pass@host/db
PRIVATE_KEY=-----BEGIN RSA PRIVATE KEY-----
```

### 风险场景

```
场景1：AI 看到环境变量
用户：帮我检查环境配置
AI: Read .env → 看到敏感变量 → 可能泄露

场景2：工具执行暴露
AI: Bash(printenv) → 输出所有环境变量 → 全部暴露

场景3：子进程继承
AI: Bash(node script.js) → 子进程继承所有变量 → 泄露风险
```

## 白名单设计

### 白名单原则

**只有白名单变量才能传递给 AI**


### 动态白名单

```typescript
// 用户可以添加白名单
function getUserWhitelist(): Set<string> {
  const settings = loadSettings()
  const userWhitelist = new Set(ENV_WHITELIST)

  // 添加用户自定义白名单
  if (settings.envWhitelist) {
    for (const env of settings.envWhitelist) {
      userWhitelist.add(env)
    }
  }

  return userWhitelist
}
```

## 环境变量过滤

### Bash 工具过滤

```typescript
async function bashTool(input: BashInput): BashResult {
  // 1. 构建安全环境
  const safeEnv = filterEnvironment(process.env)

  // 2. 执行命令（只传递安全变量）
  const result = await spawn(input.command, {
    env: safeEnv,
    cwd: context.cwd,
  })

  return result
}

function filterEnvironment(fullEnv: NodeJS.ProcessEnv): NodeJS.ProcessEnv {
  const whitelist = getUserWhitelist()
  const safeEnv: NodeJS.ProcessEnv = {}

  for (const [key, value] of Object.entries(fullEnv)) {
    if (whitelist.has(key)) {
      safeEnv[key] = value
    }
  }

  return safeEnv
}
```

### 模式匹配黑名单

```typescript
// 模式黑名单：包含敏感词的变量
const ENV_PATTERN_BLACKLIST = [
  /key/i,
  /secret/i,
  /password/i,
  /token/i,
  /credential/i,
  /auth/i,
  /private/i,
  /pass/i,
]

function isBlacklistedPattern(key: string): boolean {
  for (const pattern of ENV_PATTERN_BLACKLIST) {
    if (pattern.test(key)) {
      return true
    }
  }

  return false
}

function filterEnvironment(fullEnv: NodeJS.ProcessEnv): NodeJS.ProcessEnv {
  const safeEnv: NodeJS.ProcessEnv = {}

  for (const [key, value] of Object.entries(fullEnv)) {
    // 跳过黑名单
    if (ENV_BLACKLIST.has(key)) {
      continue
    }

    // 跳过模式黑名单
    if (isBlacklistedPattern(key)) {
      continue
    }

    // 只保留白名单
    if (getUserWhitelist().has(key)) {
      safeEnv[key] = value
    }
  }

  return safeEnv
}
```

## printenv 命令拦截

### 拦截敏感命令

```typescript
const SENSITIVE_ENV_COMMANDS = [
  'printenv',
  'env',
  'set',
  'export',
  'echo $',
]

function isSensitiveEnvCommand(command: string): boolean {
  const normalized = command.trim().toLowerCase()

  for (const sensitive of SENSITIVE_ENV_COMMANDS) {
    if (normalized.includes(sensitive.toLowerCase())) {
      return true
    }
  }

  return false
}

async function bashTool(input: BashInput): BashResult {
  // 检查是否尝试获取环境变量
  if (isSensitiveEnvCommand(input.command)) {
    return {
      success: false,
      output: '',
      error: '禁止直接获取所有环境变量。请使用安全方式查询特定变量。',
    }
  }

  // ... 正常执行
}
```

### 安全替代

```typescript
// 提供 API 查询特定白名单变量
async function getEnvVar(key: string): string | null {
  const whitelist = getUserWhitelist()

  if (!whitelist.has(key)) {
    // 不在白名单，拒绝
    return null
  }

  return process.env[key] || null
}

// AI 可以安全查询白名单变量
const path = await getEnvVar('PATH')
const home = await getEnvVar('HOME')
```

## .env 文件处理

### 不读取 .env

```typescript
// .env 文件可能包含敏感信息
const ENV_FILES = [
  '.env',
  '.env.local',
  '.env.development',
  '.env.production',
  '.env.*.local',
]

function isEnvFile(path: string): boolean {
  const filename = basename(path)

  for (const envFile of ENV_FILES) {
    if (filename === envFile || filename.match(envFile.replace('*', '.*'))) {
      return true
    }
  }

  return false
}

async function readFile(path: string): ReadResult {
  // 检查是否是 .env 文件
  if (isEnvFile(path)) {
    // 特殊处理
    return {
      success: false,
      error: '检测到 .env 文件，可能包含敏感信息。请明确指定需要读取的内容。',
      warning: true,
    }
  }

  // ... 正常读取
}
```

### 安全读取 .env

```typescript
// 如果用户明确要求读取 .env
async function readEnvFile(path: string, userConfirmed: boolean): ReadResult {
  if (!userConfirmed) {
    return {
      success: false,
      error: '读取 .env 文件需要用户确认',
    }
  }

  // 读取并过滤
  const content = await fs.readFile(path, 'utf-8')
  const filtered = filterEnvFileContent(content)

  return {
    success: true,
    content: filtered,
    warning: '已过滤敏感变量，只显示白名单变量',
  }
}

function filterEnvFileContent(content: string): string {
  const lines = content.split('\n')
  const filteredLines = []

  for (const line of lines) {
    const match = line.match(/^([^=]+)=(.*)$/)

    if (match) {
      const key = match[1].trim()

      // 只保留白名单
      if (getUserWhitelist().has(key)) {
        filteredLines.push(line)
      } else {
        filteredLines.push(`${key}=[FILTERED]`)
      }
    } else {
      // 非变量行保留
      filteredLines.push(line)
    }
  }

  return filteredLines.join('\n')
}
```
