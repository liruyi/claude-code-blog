本系列完整介绍了 Claude Code 的三大扩展机制：Skills、Hooks、MCP。本篇总结整个体系，并提供从新手到专家的学习路径。

## 三大机制对比

| 特性 | Skills | Hooks | MCP |
|------|--------|-------|-----|
| **核心概念** | 提示模板 | 事件监听 | 外部工具协议 |
| **触发方式** | 用户 `/skill-name` | 自动触发 | AI 决定调用 |
| **主要用途** | 封装操作流程 | 验证、通知、记录 | 接入外部系统 |
| **配置位置** | `.md` 文件 | `settings.json` | `.mcp.json` |
| **复杂度** | 低 | 中 | 高 |
| **用户交互** | 主动 | 被动 | 混合 |
| **可阻止操作** | ❌ | ✅（PreToolUse） | ❌ |
| **修改输入** | ❌ | ✅（PreToolUse） | ❌ |
| **提供资源** | ❌ | ❌ | ✅（Resources） |
| **提供提示** | ✅（content） | ✅（prompt） | ✅（Prompts） |

| 需求 | Skills | Hooks | MCP |
|------|:------:|:-----:|:---:|

## 组合使用建议

| 场景 | 推荐组合 |
|------|----------|
| 标准化流程（commit、review） | Skill 单独 |
| 安全验证（危险命令） | Hook 单独 |
| 接入第三方服务 | MCP 单独 |
| 审批流程（PR review） | Skill + Hook + MCP |
| 自动化部署 | Skill + Hook |
| 知识库查询 | MCP + Skill |
| CI/CD 触发 | Hook 单独 |

## 从新手到专家

### 第 1 周：基础

**目标**：理解三大机制，编写第一个 Skill

1. 阅读 01-开篇.md，理解扩展体系
2. 阅读 02-skills基础.md
3. 编写第一个 Skill：`/review`
4. 阅读 04-hooks入门.md
5. 配置第一个 Hook：日志记录

**练习**：
- 创建 3 个常用 Skill（review、commit、test）
- 配置 PostToolUse Hook 发送桌面通知

### 第 2 周：进阶

**目标**：深入 Hooks 和 MCP

1. 阅读 03-skills进阶.md
2. 编写 Fork 模式 Skill
3. 阅读 05-hooks实战.md
4. 实现 PreToolUse 验证 Hook
5. 阅读 06-mcp基础.md
6. 配置第一个 MCP Server（Slack 或 GitHub）

**练习**：
- 编写阻止危险命令的 Hook
- 配置 Slack MCP Server 并发送消息

### 第 3 周：开发

**目标**：开发自定义 MCP Server

1. 阅读 07-mcp开发.md
2. 用 TypeScript 或 Python 开发 MCP Server
3. 阅读 08-mcp资源提示.md
4. 实现 Resources 和 Prompts
5. 测试 MCP Server

**练习**：
- 开发知识库 MCP Server（搜索 + 添加文档）
- 实现 kb://index Resource

### 第 4 周：组合

**目标**：组合三大机制

1. 阅读 09-组合扩展.md
2. 实现 Skill + Hook 组合
3. 实现 Skill + MCP 组合
4. 阅读 10-企业案例.md
5. 实现完整审批流程

**练习**：
- 实现 PR 审批流程（Skill + Hook + MCP）
- 配置审计日志 Hook

### 第 5 周：优化

**目标**：性能和安全优化

1. 阅读 11-最佳实践.md
2. 优化 Hook 执行时间
3. 实现企业安全策略
4. 解决常见问题

**练习**：
- 实现 Hook 输入验证
- 配置 MCP Server 白名单
- 编写组合审计日志

## 本系列总结

本系列从扩展体系概览开始，逐步深入：

1. **Skills**：最简单的扩展，封装操作流程
2. **Hooks**：生命周期注入，验证和通知
3. **MCP**：外部工具协议，连接企业系统
4. **组合**：三大机制协同，完整自动化
5. **实践**：调试、性能、安全、踩坑

希望这个系列帮助你掌握 Claude Code 的扩展开发，构建强大的自动化工作流。
